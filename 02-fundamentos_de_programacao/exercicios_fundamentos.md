## Exercício 1:
**Crie um programa que:**
* Peça ao usuário: nome, idade e altura.
* Guarde esses valores em variáveis.
* Mostre uma mensagem assim:

```"Olá, NOME! Você tem IDADE anos e mede ALTURA metros."```
* Use conversão de tipos para idade (int) e altura (float).


## Exercício 2:
Usando uma constante PI e uma variável para o raio, faça um programa que:
* Defina PI = 3.14159.
* Peça ao usuário o valor do raio de um círculo.
* Calcule a área usando a fórmula:

`área = PI * raio²`

* Mostre o resultado na tela.

## Exercício 3 
Faça um programa que:
* Peça a idade de uma pessoa.
* Pergunte se ela é estudante (responder "s" para sim ou "n" para não).
* Considere que a pessoa tem direito a desconto se:
  * for menor de 18 ou for estudante.

* Mostre na tela:
  * `True` se ela tem desconto.
  * `False` caso contrário.


Use operadores relacionais (`<`, `==`) e lógicos (`and`, `or`, `not`).

## Exercício 4:

Crie um programa para uma lojinha que:
* Peça o nome do produto.
* Peça o preço (em texto, ex: "10.50").
* Peça a quantidade (em texto, ex: "3").
* Converta preço para float e quantidade para int.

Calcule o total da compra: `total = preço * quantidade`.

Mostre uma mensagem como:

`"Você comprou QUANTIDADE x PRODUTO. Total: R$ TOTAL"`
