# Conhecendo a linguagem Python
Python é uma linguagem de programação de alto nível, amplamente utilizada em diversas áreas da tecnologia. Criada por Guido van Rossum e lançada em 1991, seu design prioriza a legibilidade do código, utilizando uma sintaxe clara e concisa que se assemelha muito à linguagem natural, facilitando o aprendizado e a manutenção.


Originalmente concebida para ser uma sucessora da linguagem ABC, Python evoluiu para se tornar uma das linguagens mais populares e influentes do mundo, sendo aplicada em desenvolvimento web, análise de dados, inteligência artificial, machine learning, automação, desenvolvimento de jogos e muito mais.
A linguagem Python se tornou uma das mais famosas e preferidas por desenvolvedores iniciantes e experientes devido a uma série de fatores:

- **É fácil de entender, ler e aprender**: Sua sintaxe limpa e organizada requer menos linhas de código para realizar tarefas complexas em comparação com outras linguagens.
- **É uma linguagem de alto nível**: Está mais próxima da linguagem humana do que do código de máquina, o que a torna mais intuitiva para programar.
- **É multiplataforma**: O código Python pode ser executado em diferentes sistemas operacionais (Windows, macOS, Linux, etc.) com pouca ou nenhuma alteração.
- **Tem uma vasta biblioteca-padrão e um ecossistema rico**: Possui um conjunto robusto de módulos e funções já prontas, além de milhares de bibliotecas de terceiros (como NumPy, Pandas, Django) que estendem suas capacidades.
- **Python é uma linguagem de tipagem dinâmica**: Não exige que o desenvolvedor declare explicitamente o tipo de dado que será armazenado em uma variável antes de usá-la; o tipo é inferido durante a execução.
- **É uma linguagem interpretada**: O código-fonte é executado linha por linha por um interpretador (em vez de ser compilado de uma só vez). Se houver um erro, a execução é interrompida naquele ponto, facilitando a identificação e correção de bugs.
- **Possui indentação obrigatória para marcação de blocos**: Ao invés de usar chaves ou palavras-chave para definir blocos de código (como funções, laços ou condicionais), Python utiliza a indentação (espaços ou tabs), o que força o código a ser visualmente estruturado e legível.



## Primeiros passos para programar em Python
Para iniciar sua jornada na programação Python, você precisará de duas coisas principais: a própria linguagem e um ambiente confortável para escrever e executar seu código.

### Passo 1: instalação do python
1. Acesse o Site Oficial: Vá para o site [python.org](https://www.python.org).
2. Baixe a Versão Mais Recente: A versão mais atualizada para download geralmente é exibida em destaque no topo da página inicial (na seção "Downloads"). 
3. Clique no botão para baixar o instalador compatível com seu sistema operacional (Windows, macOS ou Linux).

### Passo 2: encontrando sua IDE
Uma IDE (Integrated Development Environment - Ambiente de Desenvolvimento Integrado) é um programa onde você irá escrever, editar e gerenciar seu código de forma organizada.

1. **Visual Studio Code (VS Code)**
Baixe o VS Code: Acesse o site [VS Code](https://code.visualstudio.com/download) e baixe a versão apropriada para seu sistema operacional.
   * **Instale o VS Code**: Execute o instalador baixado e siga as instruções.
   * O VS Code é genérico, então precisamos instalar uma "extensão" que o ensine a entender e trabalhar bem com o Python.
   * **Acesse as Extensões**: No lado esquerdo do VS Code, procure e clique no ícone de Extensões (geralmente se parece com quatro quadrados, um deles separado).
   * **Busque a Extensão**: Na barra de pesquisa, digite "Python".
   * **Instale a Extensão Oficial**: Procure pela extensão chamada simplesmente "Python", que é publicada pela Microsoft. Clique em Instalar.
   * **Rodando seu código python**: No terminal integrado do VS Code, você digita o comando para que o Python execute seu arquivo: ```python NomeDoSeuArquivo.py```


2. **Pycharm**
   * Baixe o Pycharm: Acesse o site [Pycharm](https://www.jetbrains.com/pycharm/) e baixe a versão apropriada.
   * Após instalá-lo, você pode criar um novo projeto a partir de uma nova pasta (open new folder) 
   * Ou apenas criar um novo arquivo (open new file).

3. **IDEs online**
   * Se você prefere não instalar nenhuma IDE no seu computador 
   * Ou apenas quer um jeito rápido de programar, 
   * As IDEs online são a solução ideal! Basta abrir o site e começar a codar.

    1. **[CodeSkulptor3](https://py3.codeskulptor.org/)**

    2. **[Online python](https://www.online-python.com/)**  
       1. Acesse o site, escreva seu código Python e clique em "Run" para executá-lo imediatamente.
   
4. **IDEs para smartphone**
* Se você deseja praticar e codar usando apenas seu celular, existem duas ótimas opções de aplicativos:

    #### Coding Python

    * **Encontre o App**: Vá até a "Play Store" (ou o gerenciador de apps do seu smartphone) e procure por "Coding Python".
    * **Identificação**: O ícone é uma tela de fundo preta com as letras "PY" no campo superior e a palavra "Coding" no campo inferior, ambas em branco.
    * Depois de baixar, abra o app. Você já pode começar a escrever seu código. Quando terminar, clique no botão "Run" para executar seu código.

    #### Pydroid 3

    * **Encontre o App**: Busque por "Pydroid 3" na Play Store (ou em seu software gerenciador de apps).
    * **Identificação**: O ícone tem um fundo azul, com um detalhe em amarelo escrito "Py³" em fonte branca
    * **Configurando o app**:
      * **Objetivo**: O sistema perguntará (em inglês) qual seu objetivo. Marque "Learning Python" e avance.
      * **Nível**: Perguntará seu nível de Python. Marque a opção "Beginner" (Iniciante) e avance
      * **Tipo de Programa**: Perguntará que tipo de programas você desenvolverá. Marque "Simple terminal or web programs" e avance.
      * **Tema**: Se desejar, configure o tema escuro e avance
      * **Ignorar Premium**: Podem aparecer anúncios ou questionamentos sobre assinaturas premium. Você pode ignorá-los clicando no botão de voltar do celular ou no **"x" / "continue"** (dependendo da interface)
      * Após a configuração, você pode começar a codar. Para rodar seu código, clique no botão de **"play" amarelo** (▶️) no canto inferior direito.
